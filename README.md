## 🌟 About the Project

VisionTrace AI v2.0 is not just a project — it is a **real-time intelligent traffic perception system** built using modern AI vision technology.

It observes roads like a human eye 👁️, understands vehicles like a brain 🧠, and transforms traffic into meaningful insights 📊.

This system is designed for **smart cities, surveillance systems, and future transportation intelligence.**

---

## ✨ Key Features

🚗 Real-time Vehicle Detection using YOLOv8  
📊 Smart Analytics Dashboard with live charts  
🎥 Image & Video AI Processing  
⚡ Fast OpenCV inference engine  
🧾 Auto report generation (CSV history)  
🚦 Intelligent traffic classification system

## 🧠 How It Works
```bash
1️⃣ Upload image or video  
2️⃣ YOLOv8 detects vehicles in real time  
3️⃣ System calculates traffic density  
4️⃣ Smart dashboard updates instantly  
5️⃣ History stored for analytics  
```
---

## 🚦 Traffic Intelligence Levels
```bash
🟢 Low Traffic → 0–15 vehicles  
🟡 Normal Traffic → 16–35 vehicles  
🟠 Heavy Traffic → 36–60 vehicles  
🔴 Very Heavy Traffic → 60+ vehicles  
```
